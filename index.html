How Many Meeples: a full stack project for logging board games.
I am a huge fan of board games, and the premier website for board games (Board Game Geek) has a counter-intuitive system for logging plays. With that in mind, I made an app that makes it easy to log plays using Ruby on Rails, Javascript, jQuery, HTML/CSS, Handlebars, and Bootstrap.

FEED Bucket: a restuarant tracking app.
When I met up to make an app with fellow developers Wei Liu and Cathy Obrien we discovered that there was nothing closer to our hearts than tasty food, so we set forward to make an app that lets you track restuarants. The app lets the user search for and log restuarants into a wish list using the Google Maps Javascript API, Express.js, MongoDB, Javascript, HTML/CSS, Handlebars, and Bootstrap.
