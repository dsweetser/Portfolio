Projects

How Many Meeples: a full stack project for logging board games.
I am a huge fan of board games, and the premier website for board games (Board Game Geek) has a counter-intuitive system for logging plays. With that in mind, I made an app that makes it easy to log plays using Ruby on Rails, Javascript, jQuery, HTML/CSS, Handlebars, and Bootstrap.

FEED Bucket: a restuarant tracking app.
When I met up with fellow developers Wei Liu and Cathy Obrien we discovered that there was nothing closer to our hearts than tasty food, so we set forward to make an app that lets you track restuarants. The app lets the user search for and log restuarants into a wish list using the Google Maps Javascript API, Express.js, MongoDB, Javascript, HTML/CSS, Handlebars, and Bootstrap.

Tic-Tac-Plaid: the marriage of two grids.
My first assignment in General Assembly's Web Development Immersive course was to create a Tic-Tac-Toe game. Being a huge fan of plaid, my mind immediately drew a connection between the two grids and the theme made the whole thing significantly more fun. The moment that I registred a click on the page to make an X on the screen was so rewarding, I knew then I had found my calling. This project was made using Javascript, HTML/CSS, jQuery, and Bootstrap.
